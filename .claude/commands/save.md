Sauvegarde les modifications sur GitHub :

1. Vérifie s'il y a des changements avec `git status`
2. Si oui, ajoute tous les fichiers modifiés avec `git add .`
3. Crée un commit avec un message descriptif basé sur les changements
4. Push sur GitHub avec `git push`
5. Confirme ce qui a été sauvegardé

Si l'argument $ARGUMENTS est fourni, utilise-le comme message de commit.
Sinon, génère un message basé sur les fichiers modifiés.
